# Projeto Dadland - Template Base Fullstack

Bem-vindo ao repositório do projeto Dadland! Este é um template base fullstack utilizando Next.js, Django e Docker.

Este arquivo serve como o mapa do projeto para orientar seus primeiros passos de forma rápida e direta.

## Roteiro de Boas-Vindas

Siga as orientações abaixo na ordem listada para engatar no desenvolvimento rapidamente:

1. **Setup Inicial e Contêineres:** 
   Abra o arquivo `.env.example` na raiz do projeto. Lá você encontrará as instruções passo a passo para criar seu `.env`, realizar o setup das configurações e subir toda a stack via Docker Compose.

2. **Entenda a Arquitetura Macro:** 
   Leia o documento [documentation.md](./documentation.md) para absorver o panorama técnico completo, como os serviços conversam entre si e as tecnologias-chave do projeto.

3. **Mergulhe nas Camadas (Guias Específicos):** 
   Na hora de colocar a mão na massa, consulte os READMEs técnicos posicionados dentro de suas respectivas pastas. Eles guardam comandos vitais, boas práticas e a estrutura de cada módulo:
   - ⚛️ **Frontend:** [Guia do Frontend (Next.js)](./dadlandfrontend/README.md)
   - 🐍 **Backend:** [Guia do Backend (Django)](./dadlandbackend/README.md)
   - 🐳 **Infraestrutura:** [Guia de Infraestrutura (Docker)](./dadlandinfra/README.md)
